根据 Microsoft 的说法，回调函数是托管应用程序中的代码，可帮助非托管 DLL 函数完成任务。对回调函数的调用间接地从托管应用程序通过 DLL 函数传递回托管实现。这个存储库包含一个回调函数列表，可用于执行与位置无关的 shellcode，以便 CreateThread 成为过去。

还有更多回调函数可以做加载器，但是具体在那个位置放shellcode，怎么个调用流程，就自行探索了
